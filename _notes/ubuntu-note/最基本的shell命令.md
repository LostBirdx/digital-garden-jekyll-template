cp  复制文件	


```shell
	cp spath dpath   复制单个文件到目标路径
	cp file1 file2 dpath    复制多个文件到目标路径
	cp spath/{file1 file2} dpath	复制spath下的多管文件到目标路径
```

mv 重命名文件

```shell
	mv A B 将目录A重命名为B
	mv /a /b/c 将目录/a 移动到/b下,并重命名为c
```
